# Frontend

A distributed parking system frontend built with React, Tailwind CSS and Vite. This web application allows users to pay for parking using their license plate number, view real-time parking capacity, and book spots in advance.

### Features

Ticketless Payment: Pay for parking by entering a license plate number.

Real-time Capacity: View available spots at various parking locations (mocked data).

Reservations: Book parking spots in advance.

Mock API Layer: Simulates latency and backend communication with Edge/Cloud actors. (only for early development stage)

### Tech Stack

Framework: React (Vite)

Styling: Tailwind CSS

Icons: Lucide React

State Management: React Hooks (useState, useEffect)

### Installation & Setup

Since this project relies on specific Node.js bindings, it is recommended to run this inside a Linux environment (native Linux or WSL2).

Install Dependencies:

```
npm install
```

Start Development Server:

```
npm run dev
```
